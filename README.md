# init-biyisport-lab

## 项目简介

`init-biyisport-lab` 是一个用于归档与发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为前端页面资源的集合与版本管理用途。

## 目录结构

```
├── pages/           # 独立 HTML 页面存放目录
├── assets/          # 共用资源文件（样式、脚本、图片等）
├── README.md        # 本文件
└── index.html       # 可选的入口页面（如有）
```

> 具体页面文件请参考 `pages/` 目录下的子文件夹或文件名说明。
